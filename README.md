# Instalador-de-musicas-
Olá, Criei esse projeto bem simples, um instalador de musicas do youtube, Voce Roda ele no seu vscode ou outra ide, e instala o ffmpeg, E ele Abaixa musicas e converte pra mp3 na pasta dowloads, Se quiser tambem crie um executavel com o pyinstaller

passo a passo. Instala o ffmpeg Link oficial: https://ffmpeg.org/download.html
no seu terminal do vscode ou ide Digite tambem: ( winget install DenoLand.Deno )
no seu terminal do vscode ou outra ide digite pip install yt-dlp

Faça um passo a passo de instalar o ffmpeg pergunte pra ia ou veja um tutorial, é bem simples

e execute o script Coloque a url do youtube e seja feliz
